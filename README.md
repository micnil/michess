## Getting started

Run `yarn`

Run `yarn nx serve webchess`

## Running unit tests

Run `yarn nx test <lib_name/app_name>` to execute the unit tests via [Jest](https://jestjs.io).

Run `nx affected:test` to execute the unit tests affected by a change.
